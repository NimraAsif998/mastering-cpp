

# Module 1: Introduction to Programming and C++

## 1. What is Programming?

Programming is the process of creating a set of instructions that tells a computer how to perform a specific task. These instructions are written using programming languages. Through programming, we can develop software applications, websites, games, and many other digital solutions.

---

## 2. Programming Languages

A programming language is a formal language used to communicate with a computer. It provides a set of rules and syntax that programmers use to write instructions. Different programming languages are designed for different purposes. Some popular programming languages include C++, Python, Java, and JavaScript.

Programming languages can be categorized into:

* Low-Level Languages
* High-Level Languages

---

## 3. Compiler vs Interpreter

A translator is software that converts a program written in a high-level language into machine language.

### Compiler

A compiler translates the entire source code into machine code at once before execution. If there are errors, they are displayed after the complete compilation process.

**Examples:** C++, C

### Interpreter

An interpreter translates and executes the source code line by line. It stops execution as soon as it encounters an error.

**Examples:** Python, JavaScript

---

## 4. Structure of a C++ Program

Every C++ program follows a specific structure. A basic C++ program consists of:

* Header Files
* Namespace Declaration
* Main Function (`main()`)
* Statements and Expressions
* Return Statement

The `main()` function is the entry point of every C++ program, and execution always starts from this function.

---

## 5. Installing Compiler and IDE

Before writing and running C++ programs, we need to install a compiler and an Integrated Development Environment (IDE).

* A **Compiler** converts C++ source code into machine code.
* An **IDE** provides tools for writing, compiling, running, and debugging programs.

Popular choices for C++ development include:

* Visual Studio Code
* Code::Blocks
* Visual Studio
Popular Compiler For C++ is:
* Mingw
