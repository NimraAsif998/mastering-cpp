## Question 2 – Calculate Total Cost

Write a C++ program to:

- Declare two `int` variables named `itemPrice` and `shippingCost`, and assign values to them.
- Declare an `int` variable named `sum`.
- Calculate the total cost by adding `itemPrice` and `shippingCost`.
- Store the result in `sum`.
- Print the total cost
